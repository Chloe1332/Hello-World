HTML is an acronym for the phrase 'Hyper Text Markup Language'. It is used for web pages to define its structure. A HTML is broken into three parts: the start tag, the contents, and the end tag. Through this, it tells the browser which piece of text is a link, a paragraph, a heading, or anything else.
